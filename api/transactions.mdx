All endpoints on this page require `Authorization: Bearer {token}` header.
</Note>

## List transactions

```
GET /api/activityall
```

Returns a paginated list of all transactions for the authenticated user.

### Query parameters

<ParamField query="limit" type="integer">
  Number of records per page. Default: `10`.
</ParamField>

<ParamField query="offset" type="integer">
  Pagination offset.
</ParamField>

<ParamField query="transaction_type" type="string">
  Filter by transaction type: `Send Money`, `Request Money`, `Deposit`, `Withdrawal`, `Exchange`, `Merchant Payment`.
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `Success`, `Pending`, `Refund`, `Blocked`.
</ParamField>

### Response

<ResponseField name="data" type="array">
  Array of transaction objects.
  <Expandable title="Transaction fields">
    <ResponseField name="id" type="integer">Internal transaction ID</ResponseField>
    <ResponseField name="uuid" type="string">Unique transaction UUID</ResponseField>
    <ResponseField name="transaction_type" type="string">Type of transaction</ResponseField>
    <ResponseField name="subtotal" type="string">Amount before fees</ResponseField>
    <ResponseField name="charge_percentage" type="string">Percentage fee charged</ResponseField>
    <ResponseField name="charge_fixed" type="string">Fixed fee charged</ResponseField>
    <ResponseField name="total" type="string">Total amount including fees</ResponseField>
    <ResponseField name="currency_code" type="string">ISO currency code</ResponseField>
    <ResponseField name="status" type="string">Transaction status</ResponseField>
    <ResponseField name="note" type="string">Optional note attached to the transaction</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp</ResponseField>
  </Expandable>
</ResponseField>

### Example

<CodeGroup>
```bash cURL
curl "https://yourdomain.com/api/activityall?limit=20&offset=0" \
  -H "Authorization: Bearer {token}"
```

```javascript JavaScript
const response = await fetch(
  'https://yourdomain.com/api/activityall?limit=20&offset=0',
  { headers: { Authorization: 'Bearer ' + token } }
);
const { data } = await response.json();
```
</CodeGroup>

## Get transaction details

```
GET /api/transaction-details
```

Returns the full detail of a single transaction.

### Query parameters

<ParamField query="id" type="integer">
  Transaction ID.
</ParamField>

<ParamField query="transaction_reference_id" type="string">
  Unique transaction UUID (alternative to `id`).
</ParamField>

### Response

Returns a single transaction object with the same fields as the list response, plus:

<ResponseField name="sender" type="object">
  Sender user details (name, email).
</ResponseField>

<ResponseField name="receiver" type="object">
  Receiver user details (name, email).
</ResponseField>

<ResponseField name="payment_method" type="object">
  Payment method used for this transaction.
</ResponseField>

### Example

```bash cURL
curl "https://yourdomain.com/api/transaction-details?id=1234" \
  -H "Authorization: Bearer {token}"
```

## Transaction statuses

| Status | Description |
|--------|-------------|
| `Pending` | Transaction submitted, awaiting processing |
| `Success` | Transaction completed successfully |
| `Refund` | Transaction was refunded |
| `Blocked` | Transaction was blocked by admin |

## Transaction types

| Type | Description |
|------|-------------|
| `Send Money` | Transfer to another registered user |
| `Request Money` | Incoming accepted payment request |
| `Deposit` | Funds added to wallet (bank, card, crypto) |
| `Withdrawal` | Funds withdrawn from wallet |
| `Exchange` | Currency exchange between wallets |
| `Merchant Payment` | Payment to a merchant |
