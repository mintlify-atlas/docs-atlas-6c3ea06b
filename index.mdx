## Welcome to DossPanels

DossPanels is a comprehensive Laravel-based digital payment platform that enables businesses and individuals to manage financial transactions, process payments, and hold multi-currency wallets — all through a single unified platform.

## Platform capabilities

<CardGroup cols={2}>
  <Card title="Multi-currency wallets" icon="wallet">
    Users hold balances in multiple currencies simultaneously, with real-time conversion rates and instant transfers between wallets.
  </Card>
  <Card title="Payment processing" icon="credit-card">
    Accept deposits via bank transfer, Stripe, PayPal, and CoinPayments (crypto). Process withdrawals through configurable payout methods.
  </Card>
  <Card title="Merchant tools" icon="store">
    Generate QR codes, accept merchant payments, issue invoices, and manage merchant groups with granular fee configurations.
  </Card>
  <Card title="Admin panel" icon="shield">
    Full administrative control over users, merchants, currencies, fees, and permissions through a dedicated admin dashboard.
  </Card>
  <Card title="Mobile API" icon="mobile">
    Complete REST API secured with Laravel Passport OAuth2, enabling mobile app integration for all user-facing features.
  </Card>
  <Card title="Security & compliance" icon="lock">
    Google 2FA, KYC verification (identity and address proof), and role-based access control throughout.
  </Card>
</CardGroup>

## Getting started

<Steps>
  <Step title="Set up the server">
    Follow the [installation guide](/installation) to deploy DossPanels with PHP 8.1+, MySQL, Redis, and a web server.
  </Step>
  <Step title="Configure the platform">
    Set up currencies, payment methods, and fee structures in the admin panel at `/admin`.
  </Step>
  <Step title="Register your first user">
    Walk through [user registration](/user-guide/registration) and understand the KYC verification flow.
  </Step>
  <Step title="Integrate the API">
    If building a mobile app, start with [API authentication](/api/authentication) to obtain a Bearer token.
  </Step>
</Steps>
